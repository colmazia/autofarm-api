## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
## API
```
get autofarm/cache/update
to get the autofarm pools info

get autofarm/:address
to get the balance in autofarm for specific address

```
